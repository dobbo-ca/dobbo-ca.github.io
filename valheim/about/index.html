About

Valheim Helper is a filterable reference for crafting and cooking recipes. Pick a row to expand it; click any ingredient to reverse-lookup recipes that use it. Filter state is encoded in the URL so you can share filtered views.

Data

Recipe data is hand-curated in YAML inside the repository. If you spot an error or want to contribute, please open an issue or PR at dobbo-ca/web-valheim .
